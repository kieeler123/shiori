# Shiori Design System - Color (Navy Theme v1)

## 1. 기본 철학

Shiori의 기본 테마는 Navy 기반이다.
완전한 Dark가 아니라, 차분하고 지적인 Deep Navy 톤을 사용한다.

---

## 2. 역할 기반 색상 정의

### Page Background
- 역할: 전체 배경
- 기준: zinc-950 기반 navy tone

### Card Background
- 역할: 기능 단위 컨테이너
- 기준: bg-zinc-900/50

### Border
- 역할: 구분선
- 기준: border-zinc-800/60

### Primary Text
- 역할: 제목, 주요 정보
- 기준: text-zinc-100

### Muted Text
- 역할: 날짜, 보조 설명
- 기준: text-zinc-400

### Primary Action
- 역할: 생성, 저장
- 기준: accent color (추후 정의)

### Secondary Action
- 역할: 취소, 복구
- 기준: neutral tone

### Danger Action
- 역할: 완전삭제, 계정삭제
- 기준: red-500 ~ red-600

---

## 3. 금지 사항

- 페이지마다 배경색 다르게 사용 금지
- 카드 스타일 변형 금지
- 버튼 종류 임의 추가 금지
- 색상 역할 중복 정의 금지

---

## 4. 확장 계획

- Dark Theme (True Dark) 별도 설계 예정
- Light Theme 별도 설계 예정
