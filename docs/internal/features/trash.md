# Feature Spec - Trash v1

## 1. 목적

사용자가 삭제한 로그를 관리한다.

---

## 2. 기능 범위

- 로그인 사용자 데이터만 조회
- deleted_at IS NOT NULL 데이터 표시
- 최신 삭제순 정렬
- 복구 기능 (deleted_at = null)
- 완전삭제 (hard delete)
- 빈 상태 UI 제공

---

## 3. 표시 항목

- 제목
- 내용 1줄 미리보기
- 삭제 날짜
- 복구 버튼
- 완전삭제 버튼

---

## 4. v1 제한 사항

- 검색 기능 없음
- 필터 없음
- 페이지네이션 없음
- 디테일 페이지 없음
- 태그 표시 없음

---

## 5. 완료 기준

- 기능 정상 동작
- 디자인 시스템 규칙 준수
- trashRepo 분리 구현

---

## 6. 디테일 페이지를 만들지 않은 이유

휴지통의 목적은 "관리"이며,
상세 확인은 핵심 기능이 아니라고 판단.

제목 + 1줄 미리보기 + 삭제날짜로 충분하다고 결정.

다량 데이터 발생 시 필요성을 재검토할 예정.
